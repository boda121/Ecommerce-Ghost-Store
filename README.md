# Ecommerce Ghost Store

## 📌 Project Overview
**Ecommerce Ghost Store** is a full-stack e-commerce application built with **ASP.NET Core MVC** and **SQL Server**.  
The system allows managing products, orders, and customers efficiently.  
This project demonstrates **clean architecture**, proper database design, and integration with RESTful APIs.

---

## ⭐ Key Features
- 🛒 User-friendly **product catalog** with search functionality
- ⚙️ **Admin panel** for product management (CRUD)
- 📦 **Order management** and tracking
- 🔐 Authentication with **JWT** and **role-based access**
- Responsive front-end design (Razor Pages / Angular integration)
- Local database using **SQL Server**
- Clean and maintainable code structure
- Easily extendable for future features

---

## 🧰 Tech Stack
- **Backend:** ASP.NET Core MVC, C#, Web API, Entity Framework Core  
- **Frontend:** Razor Pages, HTML, CSS, JavaScript, AJAX (or Angular if integrated)  
- **Database:** SQL Server  
- **Tools:** Visual Studio, Git, GitHub  
- **Other:** JWT Authentication, Embedded systems concepts (if applicable)

---

## 📁 Project Structure (High Level)

Ecommerce-Ghost-Store/ → Project root

- ├─ Controllers/ → MVC Controllers
- ├─ Models/ → C# Models
- ├─ Views/ → Razor Pages / Views
- ├─ wwwroot/ → Static files (CSS, JS, Images)
- ├─ Data/ → Database context and migrations
- └─ Services/ → Business logic and API services
